## Requirment
1. yarn
2. npm
3. node


## Usage

1. Clone or download this repo first. <br />
2. Open your terminal or cmd and run "yarn install". <br />
3. Run "yarn start". <br />
4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
